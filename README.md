### What This Demo Covers

This video provides a walkthrough of the Healthcare Data Analytics project, showing:

*   **Data Loading & Preparation:** How the healthcare dataset is imported and cleaned.
*   **Exploratory Data Analysis:** Key insights and visualizations generated from the data.
*   **Model Training:** The process of building and training the risk prediction model.
*   **Web Application Interface:** A live look at the `webapp/` functionality in action.
*   **Results & Predictions:** Demonstrating how the model outputs are presented to users.

### Run This Yourself

To try the project demonstrated in the video:
1.  Ensure you have all dependencies: `pip install -r requirements.txt`
2.  Run the main script: `python run_all.py`
3.  Open your browser to `http://localhost:5000` (or the specified address).
